clsRegExp

Module clsDirectory clsFile clsPath clsPathDialog clsRegExp

Property

code

Public Long code

コマンド終了直後の終了コードを返す。

  • 0: 変更も異常もなく終了したことを表す。
  • 1: 異常終了したことを表す。
  • 2: 操作対象に対して何らかの変更を加えて正常終了したことを表す。

message

Public String message

コマンド終了直後の終了メッセージを返す。

path

Public String path

操作対象とするファイルのパスを指定する。

destination

Public String destination

出力先となるファイルのパスを指定する。

pattern

Public String pattern

操作対象とする文字列の検索パターンを正規表現で指定する。

replacement

Public String replacement

置換後の文字列、または挿入する文字列を指定する。

globalMatch

Public Boolean globalMatch

文字列全体を操作対象とするかどうかを指定する。

  • False: 検索パターンに最初に合致した文字列のみを操作対象とする。(既定値)
  • True: 検索パターンに合致した文字列全てを操作対象とする。

Method

selectString()

Public selectString()

検索パターンに合致する文字列を持つ行を新しいファイルへ書き出す。

deleteString()

Public deleteString()

検索パターンに合致する文字列を持つ行を削除する。

replaceSubString()

Public replaceSubString()

検索パターンに合致する部分文字列を置換して新しいファイルへ書き出す。

replaceString()

Public replaceString()

検索パターンに合致する文字列を持つ行全体を新しい文字列に置換する。

insertString()

Public insertString()

検索パターンに合致する文字列を持つ行の直前に新しい行を挿入する。

addString()

Public addString()

検索パターンに合致する文字列を持つ行の直後に新しい行を挿入する。
