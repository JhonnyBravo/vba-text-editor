clsFile

Module clsDirectory clsFile clsPath clsPathDialog clsRegExp

Property

code

Public Long code

コマンド終了直後の終了コードを返す。

  • 0: 変更も異常もなく終了したことを表す。
  • 1: 異常終了したことを表す。
  • 2: 操作対象に対して何らかの変更を加えて正常終了したことを表す。

message

Public String message

コマンド終了直後の終了メッセージを返す。

path

Public String path

操作対象とするファイルのパスを指定する。

destination

Public String destination

出力先となるファイルのパスを指定する。

Method

create

Public create()

ファイルを作成する。

delete

Public delete()

ファイルを削除する。

copy

Public copy()

ファイルをコピーする。
